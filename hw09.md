## 类与对象的区别

类是对象的抽象，而对象是类的实例。例子：动物是类，猴子是对象。

## onclick事件

```

\\恶魔被鼠标点中
onclick（devil）
{
    IF the devil is on the bank THEN
        the devil get into the boat
    ELSE
        the devil goes to the bank from the boat
    ENDIF    
}

```

## 类与对象会是动词吗？

答：类与对象也可以是动词，例如，类是运动，对象是登山、赛艇、游泳、踢足球等。
